## Что делает приложение
Помогает сотруднику собрать корректное письмо в IT: выбрать тип проблемы, заполнить
только нужные поля, получить готовую тему и текст письма, а также сразу увидеть
подходящие инструкции из Wiki (чтобы часто решить вопрос без письма).

## Для кого
Для обычных пользователей (заполняют заявку) и для сотрудников IT (вкладка
«🔧 IT: обновление Wiki» — обновление базы инструкций).

## Как запустить
Открыть файл `Заявка_в_IT_отдел_v0.27_20260601_1500.html` двойным кликом в браузере.
Установка и интернет не требуются.

## Как пользоваться (пользователю)
1. Выбрать город, юрлицо, программу/базу, контакт.
2. В списке «Ваша проблема» выбрать тип (список теперь по алфавиту).
3. Заполнить поля проблемы и описание (что случилось).
4. Справа появятся «Подходящие инструкции» — сначала загляните туда.
5. Нажать «Сформировать», скопировать тему и текст, прикрепить файлы вручную.

## Форматы данных
- База инструкций встроена в сам HTML (внутри объекта APP).

## Ограничения
- Письмо приложение само не отправляет; файлы и архив прикрепляются вручную.
- Скачивание .py зависит от браузера; при блокировке скачивания скрипт будет
  скопирован в буфер обмена (вставить в новый файл вручную).
