### Что изменено
- Версия в шапке вынесена в `id="headerVersion"` и подставляется из APP функцией
  `applyHeaderVersion()` (вызов в `init()`).
- Во вкладку IT встроены три скрипта (base64) + функции
  `copyEmbeddedScript`/`downloadEmbeddedScript`/`decodeEmbeddedScript` и блок UI с
  кнопками «Скопировать скрипт» / «Скачать .py».
- Добавлены `getPinnedInstructions()` и `mergePinned()`; проблеме деклараций
  добавлено поле `pinnedLinks`; `findInstructions()` ставит закреплённые ссылки
  первыми без дублей.
- `localProblems` сортируется по `title` (русская локаль) сразу после создания.
- Безопасный запуск `bootSafe`: `init()` в try/catch + самопроверка списка, при
  сбое — видимая плашка.
- Версия поднята v0.26 → v0.27, дата обновлена.

### Что НЕ изменялось
- Бизнес-логика заявки, формирование темы письма (включая ветку «Глобус Магазин»),
  база инструкций (539 записей), дизайн, парольная логика вкладки IT.
- Логика учёта описания в подборе (п.5) — уже работала корректно, не трогали.

### Почему изменено
По прямым просьбам пользователя; правки точечные, без переписывания приложения.

### Что проверено
- Баланс тегов 1/1, `</html>` на месте, `node --check` — JS валиден.
- Прогон в jsdom: шапка = APP-версия; 42 пункта по алфавиту; декларации — корневая
  ссылка первой (с описанием тоже); «Контур Диадок» — релевантные инструкции;
